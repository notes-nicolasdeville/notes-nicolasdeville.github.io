Python Library: pyfiglet

ascii art text generator

07 Aug 2023 | in python

Table of Contents
  • Getting Started
  • Usage
  • Additional fonts

pyfiglet is a full port of FIGlet (http://www.figlet.org/) into pure python. It takes ASCII text and renders it in ASCII art fonts.

Library resources
PyPI ---
Github https://github.com/pwaller/pyfiglet
Documentation ---

Getting Started

pip install pyfiglet

Usage

from pyfiglet import Figlet
f = Figlet(font='slant')
print f.renderText('text to render')

Additional fonts

testing with:

fonts = [
    'slant',
    'small',
    'shadow',
    'rounded',
    'roman',
    'drpepper',
    'cybermedium',
    'cricket',
    'colossal',
    'chunky',
    'broadway',
    'big',
    'basic',
    'banner',
]

fonts = sorted(fonts)

from pyfiglet import Figlet

for font in fonts:
    f = Figlet(font=font)
    ascii_art = f.renderText(font)
    print(ascii_art)
