Gitignore

02 Nov 2022 | in python

04 Sep 2022

When working with Git, you do not want to sync all files to your Github repo.

Adding a file named .gitignore (no name, just starts with dot) at the root of your synced repo enables to discard files matching the patterns listed from being synced to Github.

Here is my gitignore file for my Python projects - can't remember where I found the original template so source unknown.
