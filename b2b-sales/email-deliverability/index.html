Email Setup

Import to configure SPF, DKIM & DMARC properly:

  • setup SPF
  • setup DKIM
  • setup MX records
  • setup DMARC (Domain-based Message Authentication, Reporting, and Conformance)
  • setup Custom Tracking Domain (optional - if email tracking is important to you)

Check that your technical setup has been done properly.
You can use tools like MXtoolbox.

This can be all a bit technical - I recommend using a Fiverr gig to get it done properly & quickly.
