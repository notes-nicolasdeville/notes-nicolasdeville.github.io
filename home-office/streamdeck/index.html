Tricks

use a macOS app icon as key picture

  • in the Applications folder on macOS, locate the .app of the app
  • right-click and select Show Package Contents

  • navigate to the subfolder Resources and find the icon.ics file (or the relevant .ics file)

  • drag it to the icon area of your key/workflow in the Streamdeck app
