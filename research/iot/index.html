Components

equipmentPLCs

Programmable Logic Controllers.

Used to control machinery and processes in industrial settings.

Different types of data management and control systems used in industrial automation:

SCADA

Supervisory Control and Data Acquisition.

Used to monitor and control industrial processes.

DCS

Distributed Control Systems - used to monitor and control industrial processes.

MES

Manufacturing Execution Systems.

Used to manage and optimize production processes.

EAM

Enterprise Asset Management.

Software system used to track and manage physical assets within a business or organization.

Aveva PI

Specific software platform used for data collection and analysis in industrial settings, typically used for monitoring and optimizing production processes.

Protocols

Edge Services have the ability to communicate and work with various protocols commonly used in industrial and IoT (Internet of Things) applications.

MQTT

lightweight messaging protocol used for communication between devices and servers in IoT applications.

UPLC

Unified Plant-Level Controller, is a control system architecture used in industrial automation.

OPC-UA

Open Platform Communications Unified Architecture is a machine-to-machine communication protocol commonly used in industrial automation.

ModBus

Protocol used for communication between electronic devices in industrial automation.

BLE

Bluetooth Low Energy, is a wireless communication protocol commonly used in IoT devices.

CAN/BACnet

Communication protocols used in automotive and building automation respectively.

By supporting these protocols, the Edge Services can facilitate the integration of data from various sources, making it easier for applications to process and analyze data at the edge of the network.
