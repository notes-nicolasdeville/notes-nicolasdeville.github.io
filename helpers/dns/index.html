DNS

the internet's phone book

11 Mar 2023 | in helpers

Table of Contents
  • OpenDNS
  • Changes

DNS (Domain Name Servers) are like the internet's phone book.

Just like you'd use a phone book to find someone's telephone number, a computer needs a DNS server to ask the IP address for each request you make with a name (ie a website).

The 3 main DNS providers, beyond your ISPs' (Internet Service Provider) which is usually not good, are:

DNS Provider Primary Server Secondary Server
OpenDNS 208.67.222.222 208.67.220.220
Cloudflare 1.1.1.1 1.0.0.1
Google 8.8.8.8 8.8.4.4
